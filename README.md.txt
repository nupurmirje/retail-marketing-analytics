# Retail Marketing Analytics Project

## 1. Project Overview
This project analyzes retail sales data to generate actionable insights related to customer behavior, product performance, and sales trends.  
It demonstrates an end-to-end analytics workflow using Python, SQLite (SQL), and Power BI.

---

## 2. Business Objectives
1. Analyze customer purchasing behavior
2. Identify high-value customers and top products
3. Generate KPI-level summaries for decision-making
4. Present insights through executive dashboards

---

## 3. Tools and Technologies
1. Python (data cleaning, EDA, feature engineering)
2. SQL using SQLite (analytical queries)
3. Power BI (interactive dashboards)
4. Jupyter Notebook
5. Git and GitHub

---

## 4. Project Modules
1. Python Analysis – data preprocessing and EDA  
   (Refer to `notebooks/README_PYTHON.md`)
2. SQL Analysis (SQLite) – customer and sales insights  
   (Refer to `notebooks/README_SQL.md`)
3. Power BI Dashboards – business reporting  
   (Refer to `dashboards/README_POWERBI.md`)
