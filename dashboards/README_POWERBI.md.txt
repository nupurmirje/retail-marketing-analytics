## 1. Purpose
Power BI was used to create interactive dashboards that present analytical insights in a business-friendly format.  
These dashboards are intended for managers, executives, and non-technical stakeholders.

---

## 2. Dashboards Included
The following dashboards were created:
1. Executive Overview Dashboard
2. Customer Analytics Dashboard
3. Product Performance Dashboard

---

## 3. Features Implemented
The Power BI dashboards include:
1. Calculated columns and DAX measures
2. KPI cards and trend analysis
3. Interactive slicers and filters
4. Visual breakdowns by product category and customer segments

---

## 4. Data Handling Note
All calculated columns and measures were created within Power BI using DAX.  
The original CSV datasets remain unchanged and are treated as source data.

---

## 5. Outputs
The dashboards are shared as:
1. PDF exports for easy viewing
2. Image files for quick reference and previews
