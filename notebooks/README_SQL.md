# SQL (SQLite) Analysis – Retail Marketing Analytics

## 1. Purpose
This module uses SQL via SQLite to perform analytical queries on processed retail sales data.  
The focus is on customer-level insights, revenue analysis, and KPI generation.

---

## 2. Analysis Performed
1. Customer-level revenue and order analysis
2. Identification of top customers by sales
3. Aggregated KPIs for reporting
4. Query-based summaries exported for dashboards

---

## 3. Files and Outputs
1. `06_sqlite_sql_analysis.ipynb` – SQLite queries executed via Python
2. `sql_customer_summary.csv` – customer-level metrics
3. `sql_top_customers.csv` – top customers by revenue

---

## 4. SQL Concepts Used
1. GROUP BY and aggregations
2. Filtering and ordering
3. Joins and subqueries
4. Analytical ranking

---

## 5. Business Relevance
The SQL outputs support customer segmentation and revenue-focused decision-making.
